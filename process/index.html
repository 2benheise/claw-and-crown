Cabinet Configurator

This is a visual cabinet configurator for Claw&Crown as an interactive tool, that allows someone to design and customize cabinets in real time. It provides a visual representation of cabinets with options to modify aspects such as materials, colors, layouts, doors, and finishes.

58"/40"
Codebases are separate from one another, meaning you'll need to maintain two distinct libraries. The primary difference between them lies in the battleship logic.

1x1 Drawer

This is how the grid overlay looks out of the box—nothing extra needed on your end.

2x2 Drawer

When clicking

Simply click anywhere inside the grid to add as many 2x2 drawers as the configuration allows.

All directional logic is taken care of—you don’t need to worry about it.

8x1 Drawer

When clicking

Simply click anywhere inside the grid to add as many 8x1 drawers as the configuration allows.

Only one 8x1 drawer can be added per row.

An 8x1 can be offset left or right, depending on the allowed configuration—up to 8 positions within a 58" width. Any grid squares to the right of the 8x1 drawer will be ignored.

In a 40" configuration, the 8x1 drawer can only be placed starting from the first grid position in each row.
All directional logic is taken care of—you don’t need to worry about it.

2x2/8x1 Toggle/Clear Text

  • Look for the 💎 icon to toggle the "Slab Showcase" text layer on or off.
  • Click the X icon to remove or clear the toggle text layer.
  • Once the 💎 icon has been toggled, the 📦 icon can be used to enable or disable the "Slab Storage" text layer.
  • Click the X icon to remove or clear the toggle text layer.

1x1/2x2 Toggle Magnet Security

  1. Toggle the magnetic 🔒 icon by clicking anywhere inside the 1x1.
  2. To toggle the magnetic 🔒 icon, position your mouse in the bottom right of the 2x2 and click.
  3. Press "Clear All" to clear the magnetic 🔒 icons from all 1x1 and 2x2 tiles.
  4. Restart the process by reloading the browser.
When you click the clear button for Disable 2x2/8x1, it leaves 1x1 magnetic 🔒 icons untouched.

What browsers are supported?

Extensive testing was conducted on Google Chrome for macOS, and it is recommended to use this browser for the best experience.

Have a 🐞 bug/error?

  1. Check your internet connection
  2. Are you using Google Chrome browser?
  3. Update your Google Chrome browser/OS
  4. Clear your browser cache
  5. Try incognito/private mode
  6. Disable browser extensions
  7. Try rebooting your machine
  8. If all else fails contact the SCAN3D team and send us a descript + screenshots of what went wrong

Why is the scrollbar hidden when I resize the browser window and try to scroll?

On an iPhone, scrolling is typically done using a simple touch gesture. Users swipe up or down on the screen to move through content. This gesture mimics the natural movement of fingers on a touchscreen, providing a smooth and responsive scrolling experience.

On desktop devices, this scrolling behavior is mimicked by using a mouse or trackpad. The desktop version replicates the same functionality, allowing users to scroll through content with the mouse wheel, touchpad gestures, or scrollbars, depending on the input method. The experience is designed to feel as intuitive and fluid as it would on a mobile device.
