A modern web experience for **Claw & Crown**, a brand focused on premium, handcrafted storage and display solutions for trading cards and collectibles.

🔗 **Live Site:** https://2benheise.github.io/claw-and-crown/

---

## 📌 Overview

Claw & Crown Cabinet Configurator is an interactive web application inspired by the classic board game Battleship.

It serves as a client-facing tool that allows sales representatives to configure and display cabinet layouts in real time, enabling orders to be captured and sent directly to the warehouse for production. The application allows users to arrange modular cabinet components within a structured layout, with a focus on organizing drawer configurations efficiently and intuitively.

---

## ✨ Features

### 🗂️ Product & Configuration Showcasing
- 🪵 **Wadsworth Wide Cabinet 58" + Drawer Configuration** – Premium large-format storage display  
- 🪵 **Wadsworth Wide Cabinet 40" + Drawer Configuration** – Compact alternative with modular drawers  
- 📝 **Cabinet Configurator Notes (Top-Right UI)** – Persistent contextual notes while configuring  

### 🛠️ Interactive Tools
- 📸 **Internal Screenshot Tool** – Capture configurations directly from the interface  
- ✏️ **Editable Content Blocks** – Inline editing via `contenteditable` divs for rapid iteration  

### 🚀 Performance & Structure
- ⚡ **Lightweight Frontend** – Minimal dependencies for fast load times
- 🔒 **Obfuscated JavaScript Engine** – Battleship-inspired logic protecting core functionality
